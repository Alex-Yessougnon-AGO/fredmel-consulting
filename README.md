# FREDMEL CONSULTING

Site vitrine officiel de FREDMEL CONSULTING — Cabinet de coaching et conseil basé à Abomey-Calavi, Bénin.

## 🎯 À propos

FREDMEL CONSULTING accompagne les professionnels en :
- **Leadership** — Développer votre posture de leader
- **Élite** — Personal branding et image professionnelle
- **Créativité** — Expression artistique et innovation
- **Business** — Création et développement d'entreprise

## 🛠 Technologies

- HTML5 sémantique
- CSS3 custom properties
- JavaScript Vanilla
- Design system "Éditorial Prestige"
- Intégrations : Calendly, Google Maps

## 🚀 Déploiement sur Vercel

### Via le site Vercel (recommandé)
1. Pusher le code sur GitHub
2. Aller sur https://vercel.com
3. Cliquer "New Project"
4. Importer depuis GitHub
5. Framework Preset : **Other**
6. Cliquer "Deploy"

### Via CLI
```bash
npm install -g vercel
vercel login
vercel
```
